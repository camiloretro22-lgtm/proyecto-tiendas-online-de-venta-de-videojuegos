<h1> 🎮 Análisis de Ventas de Videojuegos </h1>

<h1> 📌 Descripción del Proyecto </h1>

Este proyecto analiza un dataset histórico de ventas de videojuegos a nivel global con el objetivo de identificar patrones de consumo, tendencias del mercado y factores que influyen en el éxito comercial de un videojuego.

A través de técnicas de análisis exploratorio (EDA), limpieza de datos y validación estadística, se generan insights accionables que pueden apoyar la toma de decisiones estratégicas en la industria gaming.

<h1> 🎯 Objetivo </h1>

El objetivo principal es responder preguntas clave de negocio:

¿Qué plataformas generan mayores ingresos?

¿Cómo varían las preferencias por región (NA, EU, JP)?

¿Qué impacto tienen las reseñas (críticas y usuarios) en las ventas?

¿Qué géneros y clasificaciones (rating) son más rentables?

👉 Todo esto enfocado en optimizar decisiones de inversión, marketing y desarrollo de productos.

<h1> 🛠️ Tecnologías Utilizadas </h1>

Python (análisis y procesamiento)

Pandas & NumPy (manipulación de datos)

Matplotlib & Seaborn (visualización)

SciPy (pruebas estadísticas)

Jupyter Notebook (entorno de trabajo)
