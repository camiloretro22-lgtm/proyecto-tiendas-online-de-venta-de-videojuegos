# proyecto-tiendas-online-de-venta-de-videojuegos

Este proyecto analiza datos históricos de ventas, plataformas y reseñas de videojuegos en todo el mundo para identificar los patrones que determinan el éxito comercial de un título. El objetivo es proporcionar recomendaciones basadas en datos para planificar las campañas publicitarias de la tienda Ice para el año 2017.
