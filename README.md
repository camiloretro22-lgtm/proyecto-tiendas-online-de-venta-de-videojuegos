<h1> 🎮 Análisis de Ventas de Videojuegos </h1>

<h1> 📌 Descripción del Proyecto </h1>

Este proyecto analiza un dataset histórico de ventas de videojuegos a nivel global con el objetivo de identificar patrones de consumo, tendencias del mercado y factores que influyen en el éxito comercial de un videojuego.

A través de técnicas de análisis exploratorio (EDA), limpieza de datos y validación estadística, se generan insights accionables que pueden apoyar la toma de decisiones estratégicas en la industria gaming.

<h1> 🎯 Objetivo </h1>

El objetivo principal es responder preguntas clave de negocio:

¿Qué plataformas generan mayores ingresos?

¿Cómo varían las preferencias por región (NA, EU, JP)?

¿Qué impacto tienen las reseñas (críticas y usuarios) en las ventas?

¿Qué géneros y clasificaciones (rating) son más rentables?

👉 Todo esto enfocado en optimizar decisiones de inversión, marketing y desarrollo de productos.

<h1> 🛠️ Tecnologías Utilizadas </h1>

Python (análisis y procesamiento)

Pandas & NumPy (manipulación de datos)

Matplotlib & Seaborn (visualización)

SciPy (pruebas estadísticas)

Jupyter Notebook (entorno de trabajo)

<h1> 🔍 Proceso de Análisis </h1>

1. Limpieza y Preparación de Datos

Estandarización de nombres de columnas

Conversión de variables (ej: user_score a numérico)

Tratamiento de valores nulos (uso de mediana en variables críticas)

Creación de variable clave: ventas globales (total_sales)

📊 Impacto:
Se logró mejorar la calidad del dataset, reduciendo inconsistencias y asegurando análisis confiables.

2. Análisis Exploratorio (EDA)

Evolución de ventas por año

Identificación de plataformas dominantes

Análisis por género y clasificación ESRB

Comparación de ventas por región

📊 Hallazgo clave:
Las 5 principales plataformas concentran aproximadamente el 70% de las ventas globales, lo que indica un mercado altamente concentrado.

3. Análisis por Región

Diferencias claras entre mercados:

Norteamérica y Europa: dominadas por consolas como PlayStation y Xbox

Japón: fuerte preferencia por consolas portátiles y juegos locales

📊 Impacto de negocio:
Permite segmentar estrategias de lanzamiento y marketing por región, aumentando probabilidad de éxito.

4. Influencia de las Reseñas

Relación entre critic_score, user_score y ventas

📊 Hallazgo:

Existe una correlación positiva moderada (~0.3) entre críticas profesionales y ventas.
Las reseñas influyen, pero no determinan completamente el éxito comercial.

👉 Insight: el marketing y la marca siguen siendo factores clave.

5. Validación Estadística
Pruebas de hipótesis sobre diferencias entre plataformas y géneros

📊 Resultado:
Se identificaron diferencias estadísticamente significativas en el desempeño de ciertas plataformas, validando decisiones basadas en datos y no en intuición.

<h1> 📈 Principales Hallazgos de Negocio </h1>

📊 El mercado está altamente concentrado en pocas plataformas líderes

🌍 Existen preferencias regionales marcadas, clave para estrategias globales

⭐ Las reseñas influyen en ventas, pero no son el único factor determinante

🎮 Algunos géneros muestran mayor rentabilidad sostenida

📉 El ciclo de vida de las plataformas impacta directamente las ventas

<h1> 🚀 Conclusión </h1>

El análisis demuestra cómo los datos pueden transformarse en insights estratégicos que impactan directamente el negocio.

No se trata solo de visualizar información, sino de entender qué impulsa las ventas y cómo capitalizarlo.
